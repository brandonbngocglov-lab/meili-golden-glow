# 🏔️ 梅里雪山 · 日照金山预测系统

实时预测梅里雪山（飞来寺观景台）的"日照金山"概率。

## 数据来源
- [Open-Meteo](https://open-meteo.com/) - 免费开源气象数据（ECMWF/GFS）
- 无需 API Key，浏览器直接调用

## 预测模型
- 云量（35%）+ 天气状况（25%）+ 降水概率（20%）+ 湿度（10%）+ 季节因子（10%）
- 综合评分 0-100%，每日自动更新

## 食用方式
访问 GitHub Pages 地址即可，手机电脑均可使用。

## 技术栈
纯前端 HTML + CSS + JavaScript，单页应用，15分钟自动刷新。
